CSS Cheatsheet

Here is some CSS code I like to have on hand.

Inline-block

Used to display elements in line, but respected as blocks with a width and height.

                        .box{
                            display: inline-block;
                            width: 300px;
                            height: 300px;
                            padding: 16px;
                        }

Radial Gradient

Used to create background with color gradient in the shape of a circle.

                        .hero {
                            background-image: radial-gradient(pink, cyan, green);
                        }

Font Family

Used to establish what font style to be used.

                        .class {
                            font-family: Arial, Helvetica, sans-serif;
                        }

Padding

Creates space around elements.

                        * {
                            padding: 0px 0px 0px 0px;
                        }

Color

Makes elements that certain color.

                        #id {
                            color: goldenrod;
                        }

Comments

Allows you to comunicate with your fellow programmers without letting users see!

                        /* This make the text red */
                        .content {
                            color: red;
                        } class="explanation"
